# santander-bootcamp-excel-fiis
Este repositório mostra uma planilha de controle de investimentos em Fundos Imobiliários.

Através da plnilha é possível projetar cenários de patrimônio e dividendos considerando as taxas de juros e os aportes realizados.
É também sugerido uma porcentagem de investimento de cada segmento de FII dependendo do perfil de investidor escolhido.
